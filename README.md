# acm-template

## What is this?

给线下 ACM 比赛准备的板子。

参考了杨爷他们队的 [wf 板子库](https://github.com/fstqwq/All-in-at-the-River)

## Requirements

- `-shell-escape` in your TeX Arguments(XeLaTex).
- Pygments
- TeX Live 2022+
- Consolas, Microsoft YaHei, Cambria font.
